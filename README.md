# 게임이름

PDTG(Please Delete This Game)

## 게임 소개

이 게임은 유니티2D 횡스크롤 게임이며 마리오와 비슷한 게임이라 생각하시면 편할 듯 싶습니다.


### 게임 조작법 

왼쪽 이동: <-

오른쪽 이동: ->

점프 SpaceBar

공격:A
